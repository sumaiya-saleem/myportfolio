Prepr Network
UX/UI Designer, Prepr | May 2019 - Current

Prepr is a leading startup for open education platforms, and provides students, educators, and organizations to collaborate and solve real world challenges. In the four month internship, I was responsible for creating new solutions for improving visual design of the Prepr website and app, using Figma, Adobe Illustrator/InDesign, and InVision to wireframe, design, and prototype over 100 website and app screens.


About the Project
Two main goals of the project was to first redesign the Prepr website, and simplify content structure while adding more informational and campaign pages, keeping consistent layout and brand identity. The second goal was to add additional feature to the Prepr mobile app, and following the existing UI standards that are in place, while communicating with the outsourced development team.


Ideation
Prepr had an existing website and application in development when I joined, but was in need of design updates and addition of pages/features. When applying for the role, the company issued a design challenge which was used as a starting point to the websites redesign. Along with the design and marketing team, we created sketches and outlined a new sitemap for the website.

Style Guide
Working alongside another designer, and leading two UX designer volunteers, we implemented a style guide that would allow for consistent branding throughout the Prepr Network pages. Collaborating with other designers with this guide established the visual elements on each page. In addition, I designed numerous icons that would continue to solidify this branding.

High-Fidelity Mockups
Using Figma, we created numerous high-fidelity mockups, organizing them based on their site map hierarchy. After getting approval for each page, I assissted the developer in transorming some of these pages on Wordpress. To keep track of pages were completed, need to be revised, or were ready for development, we created an excel file to audit all the content.

Mobile Design
Using Figma, I created various screens and user flows for the mobile app. I had to work with the existing UI of the app, and had to describe the functionality of each step in the flow as this would be communicated to with an outsourced development team.
