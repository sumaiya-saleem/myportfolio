Glow Beauty: Wordpress Site
Web Design & Development, CCT460- Advanced Web Design | 2019

As a Capstone Project for an Advanced Web Design class, I was assigned to create a Wordpress Site for a mock beauty company, Glow Beauty. Tools used were Adobe Illustrator & Photoshop to create product designs, Wordpress, HTML, & CSS. The following PDF displays the process outlining the design and development stages, with screenshots of the website included at the end of the document.

Since taking this course, I work with the same professor as a teaching assistant for the first level of this course, CCT260- Introduction to Web Design, where I am responsible for aiding students with class work and assignments.
